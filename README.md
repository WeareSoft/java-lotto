# java.lotto

## [Step1] 로또 - 자동
### 기능 요구 사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.

### 프로그래밍 요구 사항
- 핵심 로직을 구현하는 코드와 UI를 담당하는 로직을 구분한다.
  - UI 로직을 `InputView`, `ResultView`와 같은 클래스를 추가해 분리한다.

> **힌트**
> - 로또 자동 생성은 `Collections.shuffle()` 메소드 활용한다.
> - `Collections.sort()` 메소드를 활용해 정렬 가능하다.
> - `ArrayList`의 `contains()` 메소드를 활용하면 어떤 값이 존재하는지 유무를 판단할 수 있다.

### 출력 결과
```
구입금액을 입력해 주세요.
14000
14개를 구매했습니다.
[8, 21, 23, 41, 42, 43]
[3, 5, 11, 16, 32, 38]
[7, 11, 16, 35, 36, 44]
[1, 8, 11, 31, 41, 42]
[13, 14, 16, 38, 42, 45]
[7, 11, 30, 40, 42, 43]
[2, 13, 22, 32, 38, 45]
[23, 25, 33, 36, 39, 41]
[1, 3, 5, 14, 22, 45]
[5, 9, 38, 41, 43, 44]
[2, 8, 9, 18, 19, 21]
[13, 14, 18, 21, 23, 35]
[17, 21, 29, 37, 42, 45]
[3, 8, 27, 30, 35, 44]

지난 주 당첨 번호를 입력해 주세요.
1, 2, 3, 4, 5, 6

당첨 통계
---------
3개 일치 (5000원)- 1개

6개 일치 (2000000000원)- 0개

```

## [Step2] 로또 - 2등
### 기능 요구 사항
- 2등을 위해 추가 번호를 하나 더 추첨한다.
- 당첨 통계에 2등도 추가해야 한다.

### 프로그래밍 요구 사항
- 예외 처리를 통해 에러가 발생하지 않도록 한다.
- 모든 원시값과 문자열을 포장한다.
- 줄여쓰지 않는다.(축약 금지)
- 일급 콜렉션을 쓴다.
